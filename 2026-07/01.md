# Day 27 - SQL Subqueries

## What I learned

A subquery is a query inside another SQL query. It is used to retrieve data that will be used by the main query.

### Simple Subquery

```sql
SELECT name
FROM employees
WHERE salary > (
    SELECT AVG(salary)
    FROM employees
);
```

### Subquery with IN

```sql
SELECT customer_name
FROM customers
WHERE customer_id IN (
    SELECT customer_id
    FROM orders
);
```

## Why Subqueries are Useful

- Break complex queries into smaller parts
- Compare data with aggregated results
- Retrieve filtered data efficiently

## Key Takeaway

Subqueries help solve complex SQL problems by nesting one query inside another.

## Practice Example

```sql
SELECT name
FROM students
WHERE marks = (
    SELECT MAX(marks)
    FROM students
);
```

# Day 17 - Introduction to SQL with Python

## What I learned

SQL is used to store, manage, and retrieve data from databases. Python can interact with SQL databases using the sqlite3 module.

### Creating a Database

```python
import sqlite3

conn = sqlite3.connect("student.db")

print("Database Created")
```

### Creating a Table

```python
import sqlite3

conn = sqlite3.connect("student.db")

cursor = conn.cursor()

cursor.execute("""
CREATE TABLE students (
    id INTEGER PRIMARY KEY,
    name TEXT,
    marks INTEGER
)
""")

conn.commit()
conn.close()
```

## Why SQL is Useful

- Stores large amounts of data
- Retrieves data efficiently
- Essential for Data Analytics

## Key Takeaway

SQL and Python work together to manage and analyze data effectively.

## Practice Example

```python
import sqlite3
```

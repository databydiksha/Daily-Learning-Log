# Day 11 - Python Modules

## What I learned

Modules are files containing Python code that can be reused in other programs.

### Importing a Module

```python
import math

print(math.sqrt(25))
```

### Importing Specific Functions

```python
from math import sqrt

print(sqrt(16))
```

## Why Modules are Useful

- Reuse code
- Organize programs
- Access built-in Python functionality

## Key Takeaway

Modules help make code modular, reusable, and easier to maintain.

## Practice Example

```python
import random

number = random.randint(1, 10)

print("Random Number:", number)
```

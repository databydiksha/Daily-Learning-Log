# Day 19 - SQL Aggregate Functions

## What I learned

Aggregate functions perform calculations on multiple rows of data and return a single result.

### COUNT()

```sql
SELECT COUNT(*) AS Total_Students
FROM students;
```

### SUM()

```sql
SELECT SUM(marks) AS Total_Marks
FROM students;
```

### AVG()

```sql
SELECT AVG(marks) AS Average_Marks
FROM students;
```

### MAX() and MIN()

```sql
SELECT MAX(marks) AS Highest_Marks,
       MIN(marks) AS Lowest_Marks
FROM students;
```

## Why Aggregate Functions are Useful

- Summarize large datasets
- Generate business insights
- Support reporting and analytics

## Key Takeaway

Aggregate functions help convert raw data into meaningful information.

## Practice Example

```sql
SELECT AVG(salary) AS Average_Salary
FROM employees;
```

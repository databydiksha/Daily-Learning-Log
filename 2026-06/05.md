# Day 1 - Python Variables and Data Types

## What I learned

Variables are used to store data in Python.

Common data types:
- Integer (int)
- Float (float)
- String (str)
- Boolean (bool)

## Example Code

x = 10
price = 99.99
name = "Diksha"
is_student = True

print(x)
print(price)
print(name)
print(is_student)

## Key Takeaway

Variables help us store and manage information in a program.

# Day 13 - Introduction to NumPy

## What I learned

NumPy is a Python library used for numerical computing and working with arrays.

### Creating a NumPy Array

```python
import numpy as np

numbers = np.array([1, 2, 3, 4, 5])

print(numbers)
```

### Basic Operations

```python
import numpy as np

numbers = np.array([1, 2, 3, 4, 5])

print(numbers + 10)
print(numbers * 2)
```

## Why NumPy is Useful

- Faster than Python lists
- Efficient for mathematical operations
- Widely used in Data Analytics and Machine Learning

## Key Takeaway

NumPy provides powerful tools for working with numerical data efficiently.

## Practice Example

```python
import numpy as np

arr = np.array([10, 20, 30])

print("Sum:", np.sum(arr))
print("Average:", np.mean(arr))
```

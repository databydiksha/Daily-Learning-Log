# Day 12 - Working with CSV Files

## What I learned

CSV (Comma-Separated Values) files are commonly used to store and exchange data.

### Reading a CSV File

```python
import csv

with open("data.csv", "r") as file:
    reader = csv.reader(file)
    
    for row in reader:
        print(row)
```

### Writing to a CSV File

```python
import csv

with open("students.csv", "w", newline="") as file:
    writer = csv.writer(file)
    
    writer.writerow(["Name", "Age"])
    writer.writerow(["Diksha", 21])
```

## Why CSV Files are Useful

- Widely used in Data Analytics
- Easy to share data
- Compatible with Excel, SQL, and Python

## Key Takeaway

CSV files are one of the most common ways to store and analyze structured data.

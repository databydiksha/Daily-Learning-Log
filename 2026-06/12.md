# Day 8 - String Operations

## What I learned

Strings are used to store text data in Python.

### Example

```python
name = "Diksha"

print(name.upper())
print(name.lower())
print(len(name))
```

### Common String Methods

- upper()
- lower()
- replace()
- split()
- len()

## Key Takeaway

String methods help manipulate and process text efficiently.

## Practice Example

```python
message = "Hello World"

print(message.upper())
print(message.replace("World", "Python"))
```

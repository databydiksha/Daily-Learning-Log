# Day 14 - Introduction to Pandas

## What I learned

Pandas is a Python library used for data analysis and manipulation.

### Creating a DataFrame

```python
import pandas as pd

data = {
    "Name": ["Diksha", "Rahul", "Priya"],
    "Age": [21, 22, 20]
}

df = pd.DataFrame(data)

print(df)
```

### Viewing Data

```python
print(df.head())
```

## Why Pandas is Useful

- Organizes data in tables
- Makes data analysis easier
- Widely used in Data Analytics and Machine Learning

## Key Takeaway

Pandas helps work with structured data efficiently using DataFrames.

## Practice Example

```python
import pandas as pd

students = {
    "Name": ["Aman", "Riya"],
    "Marks": [85, 90]
}

df = pd.DataFrame(students)
```

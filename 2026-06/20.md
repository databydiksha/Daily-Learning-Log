# Day 16 - Data Visualization with Matplotlib

## What I learned

Matplotlib is a Python library used to create charts and graphs for data visualization.

### Creating a Line Chart

```python
import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [10, 20, 15, 25, 30]

plt.plot(x, y)
plt.show()
```

### Creating a Bar Chart

```python
import matplotlib.pyplot as plt

students = ["Aman", "Riya", "Rahul"]
marks = [85, 90, 78]

plt.bar(students, marks)
plt.show()
```

## Why Matplotlib is Useful

- Visualizes trends and patterns
- Makes data easier to understand
- Widely used in analytics and reporting

## Key Takeaway

Data visualization helps transform raw data into meaningful insights.

## Practice Example

```python
import matplotlib.pyplot as plt

months = ["Jan", "Feb", "Mar", "Apr"]
sales = [100, 150, 120, 180]

plt.plot(months, sales)
plt.show()
```

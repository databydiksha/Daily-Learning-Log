# Day 24 - SQL UPDATE and DELETE

## What I learned

The UPDATE statement is used to modify existing records, while the DELETE statement removes records from a table.

### UPDATE Statement

```sql
UPDATE employees
SET salary = 60000
WHERE employee_id = 101;
```

### DELETE Statement

```sql
DELETE FROM employees
WHERE employee_id = 101;
```

### Update Multiple Records

```sql
UPDATE students
SET grade = 'A'
WHERE marks >= 90;
```

## Why UPDATE and DELETE are Useful

- Keep database records up to date
- Remove unnecessary or incorrect data
- Maintain data accuracy

## Key Takeaway

Always use the WHERE clause with UPDATE and DELETE to avoid modifying or deleting all records unintentionally.

## Practice Example

```sql
UPDATE customers
SET city = 'Pune'
WHERE customer_id = 5;

DELETE FROM customers
WHERE customer_id = 10;
```

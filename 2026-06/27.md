# Day 23 - SQL ORDER BY Clause

## What I learned

The ORDER BY clause is used to sort query results in ascending or descending order.

### Sorting in Ascending Order

```sql
SELECT *
FROM employees
ORDER BY salary ASC;
```

### Sorting in Descending Order

```sql
SELECT *
FROM employees
ORDER BY salary DESC;
```

### Sorting by Multiple Columns

```sql
SELECT *
FROM employees
ORDER BY department ASC, salary DESC;
```

## Why ORDER BY is Useful

- Organizes query results
- Makes reports easier to read
- Helps identify highest or lowest values quickly

## Key Takeaway

Use ORDER BY to sort data in ascending (ASC) or descending (DESC) order.

## Practice Example

```sql
SELECT name, marks
FROM students
ORDER BY marks DESC;
```

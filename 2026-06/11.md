# Day 7 - Tuples and Sets

## What I learned

Tuples and Sets are built-in Python data structures used to store collections of data.

### Tuples

A tuple is an ordered collection that cannot be modified after creation.

```python
fruits = ("apple", "banana", "mango")

print(fruits)
print(fruits[0])
```

### Sets

A set is an unordered collection of unique elements.

```python
numbers = {1, 2, 3, 4, 4, 5}

print(numbers)
```

## Difference Between Tuple and Set

- Tuples are ordered and immutable
- Sets are unordered and store unique values
- Tuples use ()
- Sets use {}

## Key Takeaway

Use tuples when data should not change and sets when you need unique values.

## Practice Example

```python
colors = {"red", "blue", "green", "red"}

print(colors)
```

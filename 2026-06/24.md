# Day 20 - SQL GROUP BY Clause

## What I learned

The GROUP BY clause is used to group rows that have the same values into summary rows.

### GROUP BY Example

```sql
SELECT department, COUNT(*) AS Employee_Count
FROM employees
GROUP BY department;
```

### GROUP BY with AVG()

```sql
SELECT department, AVG(salary) AS Average_Salary
FROM employees
GROUP BY department;
```

## Why GROUP BY is Useful

- Summarizes data by category
- Helps in business reporting
- Useful for dashboards and analytics

## Key Takeaway

GROUP BY combines rows with similar values and allows aggregate functions to be applied to each group.

## Practice Example

```sql
SELECT city, COUNT(*) AS Customer_Count
FROM customers
GROUP BY city;
```

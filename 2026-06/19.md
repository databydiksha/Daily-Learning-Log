# Day 15 - Data Cleaning with Pandas

## What I learned

Data cleaning is the process of fixing or removing incorrect, incomplete, or duplicate data.

### Checking Missing Values

```python
import pandas as pd

data = {
    "Name": ["Diksha", "Rahul", None],
    "Marks": [85, 90, 78]
}

df = pd.DataFrame(data)

print(df.isnull())
```

### Removing Missing Values

```python
df = df.dropna()

print(df)
```

## Why Data Cleaning is Useful

- Improves data quality
- Makes analysis more accurate
- Essential before visualization and machine learning

## Key Takeaway

Clean data leads to better insights and more reliable results.

## Practice Example

```python
import pandas as pd
```
